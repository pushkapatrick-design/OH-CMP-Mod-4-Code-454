When to Use Code 454

Critical clearance requirements
Use Code 454 when the clearance between primary and CIP is less than 6 feet at the pole or at mid-span. This code captures critical safety violations that require immediate documentation and follow-up.

Location Documentation

Specify where violation occurs
You must specify the exact location of the clearance violation: either at the pole or at mid-span between the primary and CIP conductors. This detail is essential for repair crews to locate and address the issue.

Record Measurement

Document actual clearance distance
Include the actual measured clearance distance in your report. If the clearance is 4.5 feet, document "4.5 ft" — accurate measurements help prioritize repairs based on severity.

Identify CIP Name

Include the CIP identifier
Always provide the CIP (Cathodic Inspection Program) name or identifier in your documentation. This links the clearance violation to the specific protected structure and ensures proper tracking in the system.
